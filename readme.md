# Reinforcement 5

This project implements a **UDP Client-Server** architecture in C++.

---

## How to Compile

To compile the programs, use the following commands in your terminal:

```bash
g++ udp_server.cpp -o server
g++ udp_client.cpp -o client

## How to Run

1. Start the Server first:
./server

2. Start the Client in a second terminal:
./client
